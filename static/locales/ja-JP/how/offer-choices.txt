選択可能な選択肢を表示

画面上にたくさんの選択肢を表示し、視聴者が選んだものに応じて何かを行いたい場合があります。そのためには、次の2つの要素が必要です。

• 1つ以上 @Phrase 選択可能とマークされ、一意の名前が付けられているもの（例：\Phrase('オプション1' selectable: ⊤ name: '1つ')\）• A @Choice に変化します @Phrase 最後に選択した名前。

たとえば、ここではそれぞれ一意の名前を持つ 3 つの選択肢を示します。

\option: Choice()
Group(Stack() [
		Phrase('オプション1' selectable: ⊤ name: '1つ')
		Phrase('オプション2' selectable: ⊤ name: '二')
		Phrase('オプション3' selectable: ⊤ name: '三つ')
		Phrase(option)
	])\

毎回選ばれるたびに最後に @Phrase 選択した名前が表示されます。

このパターンを使えば、選択に応じて何かを実行できます。例えば、上記のプロジェクトを変更して、選択内容に応じて秘密の絵文字を表示するようにすることができます。

\option: Choice()
option = '' ? Group(Stack() [
			Phrase('オプション1' selectable: ⊤ name: '1つ')
			Phrase('オプション2' selectable: ⊤ name: '二')
			Phrase('オプション3' selectable: ⊤ name: '三つ')
		]) Phrase({'1つ': '😡' '二': '🏀' '三つ': '❤️'}{option} ?? '？')\

その @Choice 空から始まる @Text したがって、プロジェクトの開始時にこれを確認し、3 つの選択肢を表示します。一度選択すると、それは3つの選択肢のうちの1つになります @Phrase 名前。

選択に基づいて絵文字を表示するには、 @Map 名前を秘密の絵文字に変換します。値を取得すると、 @Map 一致するものがない場合、次のように評価されます。\ø\なので、\??\デフォルトにする\'？'\オプションがマップ内にない場合。この場合、そんなことは決して起こり得ませんが、私たちはそれを知るほど賢くありません!
